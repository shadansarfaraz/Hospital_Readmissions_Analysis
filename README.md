Hospital Readmissions Analysis – Advanced Excel
Project Overview

This project analyzes hospital readmissions for diabetic patients using advanced Excel techniques, including Power Pivot, Data Model, advanced formulas, and VBA macros. The goal is to identify high-risk patients, calculate readmission risk scores, and provide actionable insights to reduce hospital readmissions, improve patient care, and optimize hospital resources.

Dataset

Source: Hospital Readmissions – Diabetes Dataset (Kaggle)

Data Description:

Patient demographics (age, gender, race)

Admissions details (admission type, length of stay, discharge disposition)

Lab procedures & medications

Diagnosis codes (ICD-9)

Readmission status (within 30 days)
